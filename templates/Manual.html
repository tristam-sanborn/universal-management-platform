Manual

Permissions

Admin

The Admin has full control over the entire system and can use all listed features below.

Users

The Users can only modify their acocunts an search for other users by default.

Accounts

Create Account

Admin can click on the create account button to create accounts by filling out all the nessesary fields.

Delete Account

An admin can delete accounts using the delete account button and verifying their action by clicking confirm.

Modify Account

Any User can modify thier own account by clicking the fields, changing the information in them, and clicking confirm.

Inventory

Create Asset

By clicking the create asset button an admin can fill out the fields and create a trackable asset.

Delete Asset

By clicking the deletre and confirm button an asset no longer needed in use can be deleted.

Modify Asset

By clicking on an asset field and updating an asset can be modified.
